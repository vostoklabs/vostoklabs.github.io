# Generator Thumbnails

Place thumbnail images here, named `<generator-id>.webp` to match the `id` field in `generators.json`.

Required images:
- `clicker.webp` — Image to Clicker generator
- `keycap.webp` — Keycap Legend Generator
- `box.webp` — Parametric Box
- `washer.webp` — Washer Generator

Recommended size: 800×500px (16:10 aspect ratio), WebP format.

If a thumbnail is missing, the card shows a styled letter placeholder.
